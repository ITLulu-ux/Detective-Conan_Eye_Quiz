Detective Conan Eye Quiz

명탐정 코난 캐릭터의 눈 이미지만 보고 정답을 맞히는 웹 퀴즈 게임입니다.

프로젝트 소개

단순 CRUD 프로젝트가 아닌, 사용자가 실제로 플레이할 수 있는 서비스를 직접 기획하고 개발하는 것을 목표로 제작하였습니다.

또한 개발에서 끝나는 것이 아니라 Docker 컨테이너화 및 AWS EC2 배포를 통해 실제 서비스 운영 환경을 경험하고자 하였습니다.

---

기획 배경

1. 실제로 동작하는 서비스를 만들어보자

기술을 학습하는 것에 그치지 않고 사용자가 직접 접속하여 플레이할 수 있는 웹 서비스를 제작하고자 하였습니다.

2. 눈만 보고 캐릭터를 구분할 수 있을까?

명탐정 코난에는 눈매가 비슷한 캐릭터들이 많습니다.

특히 모로후시 형제, 하네다 슈키치 등의 캐릭터는 눈 부분만 보면 구분하기 어렵다고 생각하였고 이를 퀴즈 게임으로 구현하였습니다.

3. 플레이 가능한 디지털 굿즈

일반적인 캐릭터 굿즈가 아닌 팬이 직접 참여할 수 있는 디지털 콘텐츠를 제작하고자 하였습니다.

4. 이스터에그

수많은 캐릭터 사이에 원작자 아오야마 고쇼를 숨겨두었습니다.

트러블슈팅

1. Docker 환경 파일 확장자 대소문자 문제

로컬 Windows 환경에서는 JPG / jpg를 구분하지 않았으나 Docker(Linux 기반) 환경에서는 대소문자를 구분하여 이미지 로딩이 실패하였습니다.

Docker 컨테이너 내부에 접속하여 파일을 확인한 결과 DB 데이터와 실제 파일명이 일치하지 않는 것을 발견하였고, DB·파일명·코드를 모두 소문자 jpg로 통일하여 해결하였습니다.

---

2. 파일명 오타 추적

일부 이미지 파일에서 meri_sera 와 merri_sera 가 혼재되어 있었습니다.

Docker 컨테이너 내부 파일 구조를 직접 확인하며 문제 원인을 추적하였고 파일명과 DB 데이터를 일치시켜 해결하였습니다.

---

3. Flask-JavaScript 비동기 처리 문제

정답 처리 시 Flask 응답을 기다리는 동안 JavaScript 애니메이션이 먼저 실행되는 문제가 발생하였습니다.

Fetch API 응답 완료 이후 후속 로직이 실행되도록 처리 순서를 제어하여 해결하였습니다.

---

4. SSH 공개키 인증 문제

AWS EC2 접속 과정에서 SSH 공개키 인증 오류가 발생하였습니다.

키 페어 생성 방식과 서버 설정을 점검하여 정상적으로 접속할 수 있도록 수정하였습니다.

---

5. PEM 권한 문제

Windows 환경에서 PEM 파일 권한 설정 문제로 SSH 접속이 거부되는 현상이 발생하였습니다.

권한 설정을 수정하여 인증 오류를 해결하였습니다.

---

6. 외부 접속 불가 문제

배포 이후 외부에서 서버 접속이 불가능한 문제가 발생하였습니다.

AWS Security Group 설정을 검토하여 필요한 포트를 개방하고 정상적으로 외부 접속이 가능하도록 구성하였습니다.
