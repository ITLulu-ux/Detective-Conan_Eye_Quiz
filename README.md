# Detective Conan Eye Quiz

명탐정 코난 캐릭터의 눈 이미지만 보고 정답을 맞히는 웹 퀴즈 게임입니다.

## 프로젝트 소개

단순 CRUD 프로젝트가 아닌, 사용자가 실제로 플레이할 수 있는 서비스를 직접 기획하고 구현하는 것을 목표로 제작했습니다.

## 기획 배경

### 1. 실제로 동작하는 서비스를 만들어보자

기술을 공부하는 것에 그치지 않고, 사용자가 직접 플레이할 수 있는 웹 서비스를 제작해보고자 했습니다.

### 2. 눈만 보고 캐릭터를 구분할 수 있을까?

명탐정 코난에는 눈매가 비슷한 캐릭터들이 많습니다.

특히 모로후시 형제, 하네다 슈키치 등의 캐릭터는 눈 부분만 보면 구분하기 어렵다고 생각했고, 이를 퀴즈 게임으로 구현했습니다.

### 3. 플레이 가능한 굿즈

일반적인 캐릭터 굿즈가 아닌, 팬이 직접 즐길 수 있는 디지털 굿즈를 만들어보고자 했습니다.

### 4. 이스터에그

수많은 캐릭터 사이에 원작자 아오야마 고쇼를 숨겨두었습니다.

## 주요 기능

* 눈 이미지 기반 캐릭터 퀴즈
* 코드네임 캐릭터 예외 처리
* 영어 이름 입력 지원
* SQLite 데이터베이스 사용
* Fetch API 기반 비동기 통신
* Docker 컨테이너 실행 지원

## 기술 스택

* Python 3.12
* Flask
* JavaScript
* Fetch API
* SQLite3
* Docker



본 프로젝트는 비상업적 학습 및 포트폴리오 용도로 제작되었습니다.
